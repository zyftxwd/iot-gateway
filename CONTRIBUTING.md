# Contributing

当前仓库以社区版源码展示和协作评审为主。

## 提交建议

- 保持改动范围清晰。
- 新增协议能力时，先补协议元数据，再补后端处理器。
- 涉及写入点位、报警、工单状态流转时，必须补充测试说明。
- 不提交生产数据库、客户数据、授权文件、现场图片和密钥。

## 代码风格

- 后端遵循现有 Spring Boot + MyBatis Plus 风格。
- 前端遵循现有 Vue 3 + Element Plus 风格。
- 新增页面要考虑权限、空状态、异常状态和大数据量展示。

## 问题反馈

提交 issue 时建议包含：

- 复现步骤
- 设备协议
- 设备配置
- 点位配置
- 后端日志
- 前端截图
- 期望结果和实际结果
